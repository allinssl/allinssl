Random Cache Plugin 示例页面

这个页面演示了插件如何为各种资源添加随机数参数

功能演示

  • CSS 文件引用（带随机数参数）
  • JavaScript 文件引用（带随机数参数）
  • 字体文件预加载（带随机数参数）
  • 已有查询参数的文件（追加随机数参数）

处理前后对比

处理前

<link rel="stylesheet" href="./styles/main.css">
<script src="./js/app.js"></script>

处理后

<link rel="stylesheet" href="./styles/main.css?v=1703123456789_abc123">
<script src="./js/app.js?v=1703123456789_def456"></script>
